# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.1.x   | :white_check_mark: |
| < 2.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of StadiumPulse Pro seriously. If you believe you have found a security vulnerability, please report it by emailing sabarnabarik@gmail.com.

Please include:
- A description of the issue
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide a timeline for resolution.
